# Note de Copie - Phase 4

Les dossiers suivants doivent être copiés depuis la racine du projet vers `phase4/` :

- `src/` → `phase4/src/` (code source complet)
- `tests/` → `phase4/tests/` (tests complets)

Ces dossiers contiennent trop de fichiers pour être créés individuellement.

**Commande pour copier (Linux/Mac) :**
```bash
cp -r src phase4/
cp -r tests phase4/
```

**Commande pour copier (Windows PowerShell) :**
```powershell
Copy-Item -Path src -Destination phase4\src -Recurse
Copy-Item -Path tests -Destination phase4\tests -Recurse
```

Une fois copiés, la Phase 4 sera complète avec tout le code source et les tests.
